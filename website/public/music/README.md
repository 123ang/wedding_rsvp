# Background music

Music files here are used by the site’s background music player.

**Your wedding tracks (play first):**
- `wedding_music.mp3`, `wedding_music_2.mp3` – main wedding music

**Summer Nude OST (added to playlist):**
- `On-The-Shore.mp3`, `One-and-one.mp3`, `Sea-house.mp3`, `Seaside-Ale.mp3`, `Something-you-want.mp3`, `Summer-Kids.mp3`, `Summer-Nude-OST-Natsu-no-yoruni.mp3`, `Sunlight.mp3`, `Sunset-Beach.mp3`, `To-Be-Continued.mp3`, `Triangle-Love.mp3`

The player tries sources in order; when one track ends, the next plays. Use the ⏭️ button to skip. Add or remove files in `src/components/BackgroundMusic.jsx` (`musicSources` array).
